# Tetris Mobile

Mobiliai pritaikytas Tetris žaidimas, skirtas atidaryti telefone per naršyklę.

## GitHub Pages

Kai įjungsi GitHub Pages, žaidimas bus pasiekiamas adresu:

```text
https://santechnika-jpg.github.io/tetris-mobile/
```

## Valdymas

- Rodyklės mygtukai ekrane: judinti ir leisti figūrą žemyn.
- `↻`: pasukti figūrą.
- `Nuleisti`: greitai numesti figūrą.
- `Pauzė`: sustabdyti arba tęsti žaidimą.
- `Iš naujo`: pradėti naują žaidimą.

## Įdiegimas telefone

Samsung telefone atidaryk žaidimo nuorodą per Chrome ir pasirink **Add to Home screen**.
